# xcsh_workload_flavor

Category: Kubernetes
Workload_flavor

## Required

- name
- namespace
- ephemeral_storage
- memory
- vcpus

## Minimal Valid Config

```terraform
resource "xcsh_workload_flavor" "example" {
  name      = "example-workload-flavor"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  vcpus             = 1
  memory            = "1024"
  ephemeral_storage = "1024"
}
```
