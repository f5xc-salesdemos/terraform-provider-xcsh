# xcsh_bigip_http_proxy

Category: BIG-IP Integration
BIG-IP HTTP Proxy in a given namespace. If one already exists, it will give an error

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "xcsh_bigip_http_proxy" "example" {
  name      = "example-bigip-http-proxy"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  advanced_profile {
  }
  disable_spec {
  }
  enable_default_profile {
  }
}
```

## Dependencies

Requires: none
