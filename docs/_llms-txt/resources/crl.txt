# xcsh_crl

Category: Certificates
Api to create crl object. configuration

## Required

- name
- namespace
- refresh_interval
- server_address
- server_port
- timeout

## Minimal Valid Config

```terraform
resource "xcsh_crl" "example" {
  name      = "example-crl"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  http_access {
  }
}
```

## Dependencies

Requires: none
