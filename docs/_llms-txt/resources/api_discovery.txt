# xcsh_api_discovery

Category: API Security
API discovery creates a new object in the storage backend for metadata.namespace

## Required

- name
- namespace

## Server Defaults (safe to omit)

- custom_auth_types

## Minimal Valid Config

```terraform
resource "xcsh_api_discovery" "example" {
  name      = "example-api-discovery"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  custom_auth_types {
  }
}
```

## Dependencies

Requires: none
