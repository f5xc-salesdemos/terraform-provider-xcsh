# xcsh_bgp

Category: Networking
Bgp object is the configuration for peering with external bgp servers. it is created by users in system namespace. configuration

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "xcsh_bgp" "example" {
  name      = "example-bgp"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  bgp_router_id = "192.168.1.1"

  bgp_peers {
    metadata {
      name = "upstream-peer"
    }
    spec {
      peer_asn     = 65000
      peer_address = "192.168.1.2"
    }
  }

  local_asn = 65001

  site {
    name      = "example-site"
    namespace = "staging"
  }
}
```

## Dependencies

Requires: none
