# f5xc_srv6_network_slice

Category: Networking
Srv6_network_slice creates a new object in the storage backend for metadata.namespace

## Required

- name
- namespace
- connect_to_access_networks
- connect_to_enterprise_networks
- connect_to_internet

## Minimal Valid Config

```terraform
resource "f5xc_srv6_network_slice" "example" {
  name      = "example-srv6-network-slice"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }
}
```

## Dependencies

Requires: none
