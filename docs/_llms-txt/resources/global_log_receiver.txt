# xcsh_global_log_receiver

Category: Monitoring
New Global Log Receiver object

## Required

- name
- namespace

## Server Defaults (safe to omit)

- ns_current

## Minimal Valid Config

```terraform
resource "xcsh_global_log_receiver" "example" {
  name      = "example-global-log-receiver"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  audit_logs {
  }
  aws_cloud_watch_receiver {
  }
  aws_cred {
  }
}
```
