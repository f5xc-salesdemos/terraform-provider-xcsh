# xcsh_malicious_user_mitigation

Category: Security
Malicious_user_mitigation creates a new object in the storage backend for metadata.namespace

## Required

- name
- namespace

## Server Defaults (safe to omit)

- mitigation_type

## Minimal Valid Config

```terraform
resource "xcsh_malicious_user_mitigation" "example" {
  name      = "example-malicious-user-mitigation"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  mitigation_type {
    rules {
      threat_level {
        high {}
      }
      mitigation_action {
        block_temporarily {}
      }
    }
  }
}
```

## Dependencies

Requires: none
