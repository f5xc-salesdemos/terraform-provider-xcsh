# xcsh_cloud_link

Category: Networking
New CloudLink with configured parameters

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "xcsh_cloud_link" "example" {
  name      = "example-cloud-link"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  aws {
  }
  aws_cred {
  }
  byoc {
  }
}
```

## Dependencies

Requires: none
