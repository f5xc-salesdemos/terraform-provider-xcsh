# xcsh_data_type

Category: Security
Data_type creates a new object in the storage backend for metadata.namespace

## Required

- name
- namespace
- is_pii
- is_sensitive_data

```terraform
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  rules {
  }

  }
  exact_values {
  }
}
```
