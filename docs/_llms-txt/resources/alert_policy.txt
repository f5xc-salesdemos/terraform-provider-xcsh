# xcsh_alert_policy

Category: Security
New Alert Policy Object

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "xcsh_alert_policy" "example" {
  name      = "example-alert-policy"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  receivers {
    name      = "slack-receiver"
    namespace = "staging"
  }

  routes {
    any {}
    send {}
  }

  notification_parameters {
    default {}
    group_wait     = "30s"
    group_interval = "1m"
  }
}
```

## Dependencies

Requires: none
