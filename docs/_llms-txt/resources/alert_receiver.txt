# xcsh_alert_receiver

Category: Monitoring
New Alert Receiver object

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "xcsh_alert_receiver" "example" {
  name      = "example-alert-receiver"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  slack {
    url = "`https://your-slack-webhook-url"`
  }
}
```

## Dependencies

Requires: none
