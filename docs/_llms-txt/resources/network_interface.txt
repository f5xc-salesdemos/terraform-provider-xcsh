# xcsh_network_interface

Category: Networking
Network interface represents configuration of a network device. it is created by users in system namespace. configuration

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "xcsh_network_interface" "example" {
  name      = "example-network-interface"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  dedicated_interface {
  }
  cluster {
  }
  is_primary {
  }
}
```

## Dependencies

Requires: none
