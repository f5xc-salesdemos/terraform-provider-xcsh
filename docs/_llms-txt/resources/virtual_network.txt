# xcsh_virtual_network

Category: Networking
Virtual network in given namespace

## Required

- name
- namespace
- legacy_type

## Minimal Valid Config

```terraform
resource "xcsh_virtual_network" "example" {
  name      = "example-virtual-network"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  legacy_type = "VIRTUAL_NETWORK_SITE_LOCAL"
}
```

## Dependencies

Requires: none
