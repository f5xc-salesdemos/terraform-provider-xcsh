# xcsh_bgp_asn_set

Category: Networking
Bgp_asn_set creates a new object in the storage backend for metadata.namespace

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "xcsh_bgp_asn_set" "example" {
  name      = "example-bgp-asn-set"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }
}
```

## Dependencies

Requires: none
