# xcsh_cloud_credentials

Category: Authentication
Api to create cloud_credentials object. configuration

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "xcsh_cloud_credentials" "example" {
  name      = "example-cloud-credentials"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  aws_secret_key {
    access_key = "AKIAIOSFODNN7EXAMPLE"
    secret_key {
      clear_secret_info {
        url = "string:///d0phbmVzc2VjcmV0a2V5"
      }
    }
  }
}
```

## Dependencies

Requires: none
