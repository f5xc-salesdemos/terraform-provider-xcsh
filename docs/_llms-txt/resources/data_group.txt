# xcsh_data_group

Category: BIG-IP Integration
Data group in a given namespace. If one already exists it will give an error

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "xcsh_data_group" "example" {
  name      = "example-data-group"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  address_records {
  }
  records {
  }
  integer_records {
  }
}
```

## Dependencies

Requires: none
