# xcsh_user_identification

Category: Security
User_identification creates a new object in the storage backend for metadata.namespace

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "xcsh_user_identification" "example" {
  name      = "example-user-identification"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  rules {
    client_ip {}
  }
}
```

## Dependencies

Requires: none
