# xcsh_cloud_connect

Category: Networking
Establishing connectivity to cloud provider networks

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "xcsh_cloud_connect" "example" {
  name      = "example-cloud-connect"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  aws_provider {
  }
  aws_tgw_site {
  }
  cred {
  }
}
```

## Dependencies
