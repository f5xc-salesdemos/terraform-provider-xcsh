# xcsh_app_api_group

Category: API Security
App_api_group creates a new object in the storage backend for metadata.namespace

## Required

- name
- namespace

## Minimal Valid Config

```terraform
resource "xcsh_app_api_group" "example" {
  name      = "example-app-api-group"
  namespace = "staging"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  elements {
  }
  bigip_virtual_server {
  }
  cdn_loadbalancer {
  }
}
```

## Dependencies

Requires: none
